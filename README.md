# 🚴 Bike Sales Analysis Dashboard (Microsoft Excel)

## 📌 Project Overview

This project is an interactive **Bike Sales Dashboard** built entirely in **Microsoft Excel**. The dashboard analyzes customer demographics and purchasing behavior to identify the factors that influence bike purchases.

Using Pivot Tables, Pivot Charts, Slicers, and Excel Dashboard techniques, the project transforms raw sales data into meaningful business insights.

---

## 🎯 Objective

The objective of this project is to:

- Analyze customer purchasing patterns.
- Identify which customer segments are more likely to purchase bikes.
- Build an interactive dashboard for business decision-making.
- Practice Excel data analysis and dashboard development skills.

---

## 🛠️ Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
- Dashboard Design
- Conditional Formatting

---

## 📊 Dashboard Features

The dashboard provides interactive analysis based on:

- Age Group
- Gender
- Marital Status
- Income
- Education
- Occupation
- Region

Users can filter the dashboard using slicers to explore different customer segments.

---

## 📈 Key Insights

Some of the insights that can be derived from the dashboard include:

- Bike purchase trends across different age groups.
- Comparison of purchases between male and female customers.
- Relationship between customer income and bike purchases.
- Regional analysis of bike sales.
- Impact of education and occupation on purchasing behavior.

---

## 📂 Dataset

The dataset contains customer demographic information such as:

- Customer ID

- Marital Status
- Education
